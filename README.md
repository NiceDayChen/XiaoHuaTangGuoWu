# 小画糖果屋

一个可直接本地打开的轻量级图片处理工具，提供图层、像素编辑和几个最核心的图片处理功能，并做成偏儿童可爱风、低门槛交互。

## 如何打开

直接双击 [index.html]即可使用。

如果浏览器限制本地文件访问，也可以在当前目录启动一个简易静态服务后再访问：

```powershell
python -m http.server 8000
```

然后打开 `http://localhost:8000`

## 已实现功能

1. 打开本地图片，支持 `jpg`、`jpeg`、`png`、`bmp`
2. 保存 PNG、另存为 PNG
3. 整图缩放、整图旋转
4. 新建图层、切换图层、隐藏图层、删除图层
5. 当前图层亮度和对比度调整
6. 当前图层色相和饱和度调整
7. 框选裁剪
8. 添加文字
9. 画笔涂鸦、直线、曲线，并可调颜色和粗细
10. 框选矩形区域并添加马赛克

## 交互说明

- 建议先打开图片，再点“新建图层”进行涂鸦、文字或马赛克处理。
- 右侧颜色调整只影响当前选中的图层。
- 裁剪、缩放、旋转会作用到整个文档和全部图层。
- 保存和另存都会导出合成后的最终 PNG 图片。
